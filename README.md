# Remate Discos V4

Next.js e-commerce application for vinyl record management with session-based authentication and queue system.

## Core Features

- Session-based authentication with device fingerprinting
- Dual language support (es-ES, en-UK)
- Real-time reservation system with queue management
- Admin dashboard for inventory and reservation management

## Tech Stack

- Next.js 14 (App Router)
- Supabase (PostgreSQL + RLS)
- TypeScript
- Zustand
- Tailwind CSS

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
